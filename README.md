# Titanic Data Analysis Project

## Objective
Perform Exploratory Data Analysis (EDA) on the Titanic dataset.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed
- Dataset Overview
- Missing Values Analysis
- Survival Analysis
- Gender Analysis
- Age Distribution
- Passenger Class Analysis

## Key Findings
- Dataset contains 891 passengers.
- 342 passengers survived.
- 549 passengers did not survive.
- Female survival rate was significantly higher than male survival rate.
- Age and Cabin columns contained missing values.
- Most passengers belonged to third class.
